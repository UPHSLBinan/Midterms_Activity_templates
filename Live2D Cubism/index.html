Freely animate the original art with Live2D

Live2D can freely animate your illustrations. All you need is an illustration separated for each part. Live2D can be used for games, videos, apps, and more.

See More Live2D Features

Extensive deformation tools

Live2D Cubism creates dynamic expressions by applying deformation to the source illustration. Multiple deformation tools are available for a wide range of cases. Select and combine deformation tools according to the application and purpose to perform modeling exactly as you imagine.

See More Live2D Features

Deform Path

Deform Path is a tool for creating dynamic deformation such as movement of hair or opening and closing of the mouth. ArtMesh can be deformed along the curve of the Deform Path. The precise degree of deformation can be set, allowing you to freely apply deformation to any location and for any purpose.

Warp Deformer

Warp Deformer is a function used for deformation of a plane, used for purposes such as flapping of clothing. It is used to produce expressions that have a feeling of depth. You can set a parent-child relationship between Deformers and move them in sync. These parent-child settings can dramatically increase modeling efficiency.

Rotation Deformer

Rotation Deformer is a function for expressing movement that rotates around a pivot point, such as when moving a neck or joint. In the same way as Warp Deformers, you can set parent-child relationships here also.

AE(After Effects) Plugin

Live2D Cubism files can be loaded into Adobe After Effects. This eliminates the need for a time-consuming and troublesome correction process when porting between software applications, and achieves high work efficiency.
